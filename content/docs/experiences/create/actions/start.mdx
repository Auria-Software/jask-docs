# Les Déclencheurs

Le déclencheur est le point de départ du scénario. Il conditionne l’exécution du parcours. 

## Les fonctionnalités

### Ouverture

Déclenchement ponctuel, lorsque vous lancez manuellement le scénario.

![experience_start](@public/experiences/open.png)

### Périodique

Déclenchement automatique à intervalles réguliers. 

Plusieurs s’options s’offrent à vous : 

- Toutes les X heures
- Toutes les X minutes
- Quotidien
- Hebdomadaire
- Mensuel

![experience_start](@public/experiences/periodic.png)
