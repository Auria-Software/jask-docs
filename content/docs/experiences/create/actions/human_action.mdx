# Les actions Utilisateurs

Ces actions apparaissent directement dans le parcours et sollicitent l’interaction. 

## Les fonctionnalités

### Afficher le menu

Vous pouvez proposez un ensemble de choix à l’utilisateur sous forme de choix sélectionnables. Cette action est idéale pour orienter le parcours selon les besoins. A chaque nouveau choix, un nouveau point de sortie est créé dynamiquement pour que vous puissiez le relier à un parcours de votre choix.

![experience_human_action](@public/experiences/menu.png)

### Envoyer un message

Permet d'affichez un message ou une information pour guider l’utilisateur. 

![experience_human_action](@public/experiences/message.png)


### Saisir une entrée

Permet de demander à l’utilisateur de fournir une information sous forme de texte. Vous pouvez configurer le message à afficher avant la saisie de l'utilisateur.

![experience_human_action](@public/experiences/input.png)

### Importer des documents

Permettez à l’utilisateur d’ajouter un ou plusieurs fichiers pour enrichir le scénario. Vous pouvez demander autant de documents que vous le souhaitez en spécifiant le nom de chaque document. A chaque nouveau document, un nouveau pin de sortie est créé dynamiquement pour que vous puissiez le relier à un parcours de votre choix par exemple une application qui va analyser le document et en extraire les informations.

![experience_human_action](@public/experiences/doc.png)

### Confirmer une opération

Sollicitez une validation “Oui / Non” avant de poursuivre le scénario. Chaque option (Oui ou Non) déclenchera donc son propre parcours distinct, permettant de gérer des suites différentes selon la réponse de l’utilisateur. Vous pouvez configurer le message à afficher avant la validation de l'utilisateur.

![experience_human_action](@public/experiences/confirm.png)
