# Le Prompts Store


Le *Prompts Store* est une bibliothèque centralisée qui vous permet de **créer, organiser et réutiliser des prompts** pour vos interactions avec l’IA. C’est un espace collaboratif pensé pour faciliter le partage de bonnes pratiques et gagner du temps dans vos recherches.

![prompts_store](@public/prompts_store/prompts_store.png)

Le Prompts Store est accessible depuis la barre de navigation principale. Vous y trouverez :

* Des prompts enregistrés par vous ou vos collègues.
* Des sections pour trier les prompts par thèmes.
* Un moteur de recherche pour retrouver rapidement le prompt que vous cherchez.




## Créer un nouveau prompt

Vous pouvez créer un prompt personnalisé pour répondre à un cas d’usage spécifique :

1. Cliquez sur le bouton **Nouveau Prompt**.
2. Donnez-lui un **titre et une description explicite**
3. Sélectionnez la visibilité de votre prompt : Privé (moi uniquement) ou Partagé (avec le reste de mon entreprise).
4. Définissez un tag pour le ranger le prompt dans la bonne section.
5. Enfin, saisissez votre prompt dans la zone de texte.

![prompts_store](@public/prompts_store/create.png)



## Utiliser un prompt enregistré

Une fois le prompt sauvegardé, vous pouvez :

* L'utiliser directement depuis toutes les barres de chat dans Jask.
* Réutiliser les prompts dans vos applications ou carnets de travail également.

![prompts_store](@public/prompts_store/use.png)

## Partager et collaborer

Le Prompts Store est conçu pour favoriser la collaboration :

* Partagez vos meilleurs prompts avec votre entreprise en mettant la visibilité sur `Partagé` au moment de la création.
* Découvrez et réutilisez les prompts créés par vos collègues.
* Améliorez progressivement la bibliothèque commune en enrichissant et en testant les prompts.
