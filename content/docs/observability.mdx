# Observabilité

Depuis les réglages, allez sur la page `Observabilité`, vous pouvez suivre la consommation des agents et applications de votre organisation.

![observability](@public/settings/observability.png)


### Filtrer sur une période

Choisissez la période sur laquelle vous voulez voir les données en haut à droite de la page.

![observability](@public/settings/observability-date.png)

Dans cette vue vous avez un résumé de la consommation globale dans la partie haute de la page. 

![observability](@public/settings/kpi-global.png)

En bas à gauche de la page, vous avez le détail de consommation en sélectionnant le type d'élément que vous voulez voir par utilisateur, par agent, par application, etc.

![observability](@public/settings/kpi-users.png)
