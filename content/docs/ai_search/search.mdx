# La Recherche IA dans vos sources de données

Depuis cette page `Documents`, vous pouvez rechercher des documents dans vos sources de données.
Commencez simplement par entrer des mots-clés ou une phrase dans la barre de recherche pour rechercher dans l'entièreté des sources de données dont vous avez accès.

![ai_search](@public/documents/screen.png)

---

Vous pouvez effectuer une recherche uniquement sur une base de données en la sélectionnant dans la liste déroulante à gauche de la page.

![ai_search](@public/documents/search_type.png)

Quand vous faites une recherche vous avez deux options possibles :

- `Recherche Rapide` : Cette option vous permet de rechercher des documents uniquement sur **les metadonnées** de ceux-ci.
- `Recherche Avancée` : Cette option vous permet de rechercher des documents sur **les metadonnées et le contenu** de ceux-ci grâce à l'IA.

--- 

Une fois la recherche effectuée, vous pouvez voir les résultats de la recherche dans la zone de résultats en dessous de la barre de recherche.

![ai_search](@public/documents/filter.png)

Vous avez la possibilité de filtrer les résultats de la recherche par `Pertinence`, `Sources`, `Type de fichiers`et `Dossiers`.

La pertinence des ressources est indiquée sur chaque document par une pastille colorée.
- `Vert` : Document très pertinent
- `Jaune` : Document pertinent
- `Rouge` : Document peu pertinent
- `Noir` : Document non pertinent



## Utiliser les résultats de la recherche

Il est possible de sélectionner des documents dans la liste des résultats et de lancer deux types d'actions

![ai_search](@public/documents/run.png)

### `Chat` : Chat avec les documents
Cette action vous permet de lancer une conversation avec les documents séléctionnés.
Vous pouvez sélectionner un modèles d'IA dans la liste déroulante des modèles auxquels vous avez accès et commencer une discussion en entrant votre question dans la zone de saisie en bas de la fenêtre principale.



### `Nouvelle Tache` : Lancer une application sur les documents
Cette action vous permet de lancer une application sur les documents sélectionnés.
Vous pouvez sélectionner une application dans la liste déroulante des applications auxquelles vous avez accès et lancer l'application.
