# Espace de Stockage Personnel

Cet espace de stockage vous est **personnel**.

Dans la limite de 5Go, vous avez la possibilité de gérer des dossiers et des fichiers afin de déposer des ressources particulières qui ne sont pas accessibles au travers des ressources partagées au niveau de l’organisation.

Ces fichiers, bien que personnels, peuvent être accessibles à vos collaborateurs si vous les liez à un Assistant que vous partagez. Néanmoins, ils ne peuvent pas être modifiés, seulement lus le cas échéant.

Il sera bientôt possible d’associer à “Mes documents” un espace OneDrive.

![image.png](@public/my_documents/image.png)

## Création et suppression d’un dossier

Vous pouvez créer un dossier en cliquant sur le bouton “+Dossier” depuis la barre de boutons en haut à droite. Il suffit de nommer votre nouveau dossier et de valider.

Pour supprimer un dossier, cliquez sur le menu “…” en regard du dossier et choisissez “Supprimer”.

Attention, la suppression est définitive et supprime également les fichiers ou sous-dossier contenus dans le dossier.

<Callout>
Pour naviguer dans les dossiers, il suffit de cliquer deux fois rapidement sur un nom de dossier.
</Callout>

## Ajout de fichiers

Pour ajouter un fichier au dossier, il vous suffit de sélectionner le bouton “Importer” depuis la barre de boutons en haut à droite. Depuis la fenêtre qui s’ouvre, naviguer sur votre disque pour sélectionner le fichier à télécharger sur JAsk et validez.

<Callout>
Un double clic rapide sur un fichier permet d’engager immédiatement une conversation avec celui-ci.
</Callout>

Il est possible de visualiser un fichier en cliquant sur “…” puis “Visualiser”.

## Actions sur les dossiers et fichiers

En sélectionnant un dossier ou un fichier vous avez accès à l’indexation ou la création d’un assistant sur le ou les éléments sélectionnés.

Une fois sélectionné l’élément choisi, cliquer sur le bouton “Exécuter”.

Les options disponibles sont :

- **Créer un assistant** : Vous avez la possibilité de créer un assistant sur ces éléments. Pour en savoir plus sur la création des assistants, veuillez vous reporter à la page dédiée [Mes assistants](https://www.notion.so/Mes-assistants-1852e06c889d80e2923fcf7536f0ce73?pvs=21)
- **Indexer le contenu** : L’indexation n’est pas automatique. Vous devez la demander spécifiquement.
    
    ![image.png](@public/my_documents/image%201.png)
    
    L’indexation prend quelques secondes, le choix de la stratégie peut également impacter le temps d’indexation (Rapide vs Approfondie) ainsi que sur les coûts.
    
    Laissez faire JAsk, il vous préviendra une fois l’indexation terminée.
    
    ![image.png](@public/my_documents/9d41b17c-ca47-49d9-b342-4da5d16e6b90.png)
    
    Une fois indexé, l’élément comporte un indicateur.
    
    ![image.png](@public/my_documents/image%202.png)
    
- **Supprimer les éléments** : Vous permet de supprimer les éléments sélectionner.

Le menu “…” au regard de l’élément vous permet également de télécharger l’élément sur votre disque dur.
