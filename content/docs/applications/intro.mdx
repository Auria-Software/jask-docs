Les applications ou workflows sont des executions de tâches prédéfinies pour automatiser des processus métiers.
Les applications sont créées par les membres de votre organisation et sont utilisables partout dans Jask.
Utilisez les applications depuis l'accueil de Jask ou depuis la page `Applications` dans le catalogue de Jask.
Si vous avez les droits nécessaires, vous pouvez créer une application depuis la page `Applications` dans le catalogue de Jask.
