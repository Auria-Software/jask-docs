# Appeler un Assistant

A la différence d'un simple prompt, l'assistant est relié à des sources de données, à un prompt système associé et a déjà un modèle d'IA sélectionné. Vous pouvez appeler un assistant dans une application pour réaliser une tâche spécifique.

![application_assistant](@public/applications/assistant.png)

Écrivez votre instruction (prompt) dans la zone de texte pour guider l'assistant à réaliser la tâche.

<Callout type="info">
Tout ce que vous reliez à gauche de l'action sera passé dans le prompt envoyé à l'assistant. Ce qui sort à droite de l'action sera uniquement la réponse de l'assistant.
</Callout>
