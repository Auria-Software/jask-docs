# Les tâches prédéfinies

![application_task](@public/applications/tasks.png)

À ce jour, il existe une seule tâche prédéfinie :

### Analyse de document Word

Cette action permet à l'IA de lire le contenu mais également tous les commentaires d'un document Word.

<Callout type="warning">
Cette action ne peut être utilisée que sur des documents Word.
</Callout>


Pour utiliser cette action, cliquez sur le bouton `+` puis sur `Tâche prédéfinie` et enfin sur `Analyser un document Word`.
Une fois l'action dans le workflow, vous pouvez y attacher le contexte où l'utilisateur déposera son document Word pour que l'app puisse y lire le contenu et les commentaires.

Écrivez vos instructions dans la zone de texte pour guider l'IA à analyser le document Word.
