# Envoyer un mail

![application_mail](@public/applications/mail.png)

## Les fonctionnalités

### Envoyer un mail
Cette action vous permet d'envoyer un mail à un ou plusieurs destinataires.

<Callout type="warning">
Les emails envoyés par cette action le sont depuis l'adresse email de votre compte de connexion à Jask.
</Callout>

![application_mail](@public/applications/send.png)

Pour utiliser cette action, cliquez sur le bouton `+` puis sur `Mail` et enfin sur `Envoyer un mail`.
Une fois l'action dans le workflow, vous pouvez entrer l'adresse email du destinataire, le sujet du mail et le contenu du mail.

<Callout type="info">
Toutes les entrées de cette action contribuent au contenu du mail envoyé.
</Callout>

### Envoyer un mail (Draft)

Cette action vous permet de créer un brouillon de mail dans votre boite mail à partir des entrées de l'action.

![application_mail](@public/applications/draft.png)

Pour utiliser cette action, cliquez sur le bouton `+` puis sur `Mail` et enfin sur `Envoyer un mail (Draft)`.
Une fois l'action dans le workflow, vous pouvez entrer l'adresse email du destinataire, le sujet du mail et le contenu du mail.

<Callout type="info">
Toutes les entrées de cette action contribuent au contenu du draft de mail.
</Callout>
