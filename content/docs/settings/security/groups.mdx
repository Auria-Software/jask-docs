# Groupes

Les groupes permettent de gérer plusieurs utilisateurs en même temps. Ils permettent notammenent d'associer des rôles à l'ensemble des utilisateurs d'un groupe ou de partager certains éléments aux utilisateurs de ce groupe.

## Création d'un groupe

Depuis Groupes, l'administrateur peut créer un nouveau groupe en cliquant sur le bouton `+ Nouveau`

![image.png](@public/settings/groups.png)

L'administrateur définit les paramètres du groupe : 
- Nom du groupe
- Propriétaire du groupe : Liste des utilisateurs qui pourront gérer ce groupe
- Membres : Liste des utilisateurs du groupe
- Rôles : Liste des rôles dont bénéficieront tous les utilisateurs du groupes

Pour plus d'information sur les rôles, consultez la page [Gestion des rôles](/security/roles.mdx).

## Édition et suppression des groupes

Vous pouvez modifier les informations d'un groupe en cliquant sur le bouton bleu d'édition sur une ligne du groupe.

Pour supprimer un groupe, cliquez sur le bouton rouge sur la ligne du groupe. Il vous faudra valider cette action avant qu'elle soit prise en compte.

![image.png](@public/settings/groups_edit.png)
