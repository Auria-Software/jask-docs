# Gestion des rôles

Les rôles dans jAsk donnent accès à des fonctionnalités ou des vues particulières. Ils s'appliquent aux comptes utilisateurs connus dans JAsk.

<Callout type="info">
Seul un utilisateur avec le droit `Utilisateurs` peut les gérer
</Callout>
