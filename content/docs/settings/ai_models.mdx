# Les Modèles d'IA

Depuis les réglages, allez sur la page `Gestion des Modèles`, vous pouvez gérer les modèles d'IA disponibles dans votre organisation.

![ai_models](@public/settings/model.png)

## Ajout d'un modèle d'IA

Commencez par cliquer sur le bouton `+` pour ajouter un modèle d'IA. Vous pouvez ajouter des modèles d'IA d'Azure, des modèles d'IA publiques ou vos propres modèles d'IA fine tuned.
Renseignez toutes les informations nécessaires pour ajouter un modèle d'IA puis testez la connexion pour vérifier que le modèle d'IA est accessible.

![ai_models](@public/settings/model-add.png)

Une fois le modèle d'IA ajouté, cliquez sur le modèle dans la liste pour voir les details de celui-ci. C'est sur cette page que vous pouvez modifier les informations du modèle d'IA, donner des droits d'accès à des utilisateurs ou supprimer le modèle d'IA.

<Callout type="info">
Quand vous partagez un modèle d'IA avec un utilisateur, celui-ci aura accès à ce choix de modèle d'IA dans toutes les applications et assistants qu'il créera si il a les droits de créer.
</Callout>

![ai_models](@public/settings/model-update.png)
