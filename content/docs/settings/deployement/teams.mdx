# Les déploiements API et Teams

Depuis les réglages, allez sur la page `Déploiements`, vous pouvez gérer les déploiements API et Teams de vos agents et applications disponibles dans votre organisation.

![deployement](@public/settings/deploy.png)


## Ajout d'un déploiement

Si vous voulez pouvoir appeler vos agents et applications via l'API, vous devez d'abord créer un déploiement API pour avoir des clés d'accès.

Commencez par cliquer sur le bouton `+` pour ajouter un déploiement. Donnez un nom à votre déploiement, sélectionnez l'agent ou l'application à déployer et choisissez le type de votre déploiement (Web (API) ou Microsoft Teams).

![deployement](@public/settings/deploy-add.png)

Une fois le déploiement créé, vous devez éditer le déploiement pour configurer les clés d'accès.

![deployement](@public/settings/deploy-update.png)

Vous pouvez créer une clé d'accès en cliquant sur le bouton `Ajouter` pour créer une clé d'accès.


## Utilisation de l'API Jask

Vous pouvez maintenant utiliser l'API Jask pour appeler vos agents et applications.
