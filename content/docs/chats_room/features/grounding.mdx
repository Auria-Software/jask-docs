# Les Sources de Réponse

Les sources de réponses sont les documents, fichiers ou autres éléments d'information que JAsk utilise pour répondre à vos questions. Nous nous efforçons de fournir des réponses précises et pertinentes en nous basant sur les sources de réponse les plus appropriées.

Vous trouverez des numéros dans les réponses de l'agent pour identifier les sources de réponses utilisées. Passez la souris sur le numéro pour voir le titre et une partie du contenu de la source de réponse.
![grounding](@public/chat_rooms/grounding.png)

<Callout>
Cliquez sur le numéro pour ouvrir le document dans l'outil d'origine de la source. (SharePoint, Salesforce, Web, etc...)
</Callout>

Les sources de réponses peuvent être de natures différentes en fonction des connaissances dont l'agent dispose :

- Documents internes
- Fichiers des connecteurs (SharePoint, Salesforce...)
- Informations extraites du web

La liste de toutes les sources de réponse est disponible tout en bas de la réponse de l'agent.
