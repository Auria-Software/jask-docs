# Les Projets

Les projets sont des espaces de travail collaboratifs pour permettre à des équipes de travailler ensemble sur certains documents dans un espace cloisonné.

![projects](@public/projects/screen.png)

## Création d'un projet

Commencez par créer un projet, entrez un nom, une description et sélectionnez la visibilité du projet. Vous avez deux autres sections dans la création `Chat Room` et `Sécurité & Gouvernance`.
Dans `Chat Room`, vous pouvez sélectionner le modèle d'IA à utiliser pour le chat avec les documents du projet.
Dans `Sécurité & Gouvernance`, vous pouvez sélectionner le type de droits d'accès aux sources.

**Mode de contrôle d'accès aux ressources**

    * **`Délégué à la source`** permet de déléguer le contrôle d'accès aux ressources à la source de données par exemple pour un agent connecté à l'entièreté d'un sharepoint si deux utilisteurs n'ayant pas les mêmes droits dans le sharepoint posent la même questions, les réponses seront différentes. En effet, l'asssitant répondra qu'avec les documents dont l'utilisateurs posant la question a accès.

    * **`Partage via Jask`** permet de partager les ressources via Jask, dans ce cas les deux utilisateurs auront la même réponse se basant sur l'entièreté des documents dont l'assistant a accès.

![projects](@public/projects/create.png)

## Importation de documents

Vous pouvez importer des documents dans le projet en cliquant sur le bouton `Importer depuis mes ressources` dans la section `Accueil` ou `Ressources`.

![projects](@public/projects/import.png)

Recherchez les documents à importer dans le projet et cliquez sur le bouton `Importer` pour les ajouter au projet.

![projects](@public/projects/add_docs.png)

## Les sections d'un projet

Vous débloquerez alors différentes sections par projet

- `Accueil` : Pour accéder aux informations du projet
- `Ressources` : Pour gérer les documents importés du projet et importer de nouveaux documents
- `Chats Room` : Pour discuter avec tous les documents du projet grâce à l'IA

![chat_room](@public/projects/chat.png)

- `Activités` : Pour garder un œil sur les activités du projet (Applications lancées dans le projet, etc.)

![activities](@public/projects/activities.png)

- `Notes` : Pour sauvegarder les résultats de vos applications et discussions avec l'IA

![notes](@public/projects/notes.png)

- `Membres` : Pour gérer les membres pouvant accéder au projet
